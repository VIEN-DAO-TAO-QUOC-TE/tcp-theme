# Share Icons

Place share button icon images in this folder with these exact file names:

- `facebook.png`
- `instagram.png`
- `x.png`
- `in.png`

Recommended size:

- 24x24 px (or larger square PNG/SVG with transparent background)

Notes:

- The single post share rail will automatically use these images when present.
- If an image is missing, it falls back to the built-in SVG icon.
